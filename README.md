# Quick Notes

Невеликий веб‑додаток для швидких нотаток на чистому HTML/CSS/JS. Нотатки зберігаються в `localStorage`, є темна тема та підтвердження видалення.

## Можливості

- **Створення / редагування / видалення нотаток**
- **Підтвердження видалення** (модальне вікно “Are you sure…?”)
- **Збереження в браузері** через `localStorage` (після перезавантаження все лишається)
- **Світла / темна тема**
- **Адаптивна верстка** (мінімально під мобільні екрани ~375px)

## Як запустити

Це статичний проєкт — бекенду немає.

### Варіант 1: просто відкрити файл

1. Відкрий `index.html` у браузері.

### Варіант 2: через Live Server (рекомендовано)

1. Встанови розширення **Live Server** у VS Code / Cursor
2. Відкрий папку проєкту
3. Запусти **Go Live** (або “Open with Live Server”) для `index.html`

## Структура проєкту

```text
quick-notes-app/
  index.html
  css/
    style.css
  js/
    script.js
  img/
    logo.svg
```

## Примітки

- Дані зберігаються локально в браузері. Щоб “обнулити” нотатки — очисти `localStorage` для цього сайту (або відкрий DevTools → Application → Local Storage).
